# CRM Comercial Pro

CRM web em Next.js + TypeScript + Tailwind CSS, com layout dark, paginas principais e dados mockados para simulacao local.

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse: `http://localhost:3000`

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Estado local e dados mockados (sem banco de dados)
